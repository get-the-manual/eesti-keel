# Kaassõnad (Предлоги и послелоги)

## Предлоги / **Eessõnad** (стоят перед словом):

### 1. Самые частые предлоги
- **Enne** (перед / до): _Enne tundi_ (Перед уроком).  
- **Pärast** (после): _Pärast tööd_ (После работы).  
- **Kesk / Keset** (посреди): _Keset tuba_ (Посреди комнаты).  
- **Mööda** (вдоль / по): _Mööda teed_ (По дороге).  
- **Ilma** (без): _Ilma minuta_ (Без меня).     

### 2. Предлоги, которые могут быть и послелогами
Некоторые слова в эстонском  могут стоять и до, и после слова:  
- **Läbi** (через / сквозь): _**Läbi** akna_ или _Akna **läbi**_ (Через окно).  
- **Üle** (через / над): _**Üle** jõe_ (Через реку).  

## Послелоги / **Tagasõnad** (стоят после слова)

| **Послелог** | **Значение**        | **Комментарий, Пример**                                                                   |
| ------------ | ------------------- | ----------------------------------------------------------------------------------------- |
| **peal**     | на (поверхности)    | **Kapi peal** on toalill (На шкафу цветок).                                               |
| kohal        | над                 | не вплотную. Peegel on kraanikausi kohal.                                                 |
| **all**      | под                 | **Laua all** ei ole midagi (Под столом ничего нет).                                       |
| **ees**      | перед               | **Akna ees** on kardinad (Перед окном шторы).                                             |
| **juures**   | у / около           | **Akna juures** on laud (У окна — стол).                                                  |
| **vahel**    | между               | **Voodi ja tooli vahel** (Между кроватью и стулом).                                       |
| keskel       | в центре            | Toa keskel on laud (В центре комнаты стол)                                                |
| kõrval       | рядом               | минимальная дистанция. Pilt on akna kõrval seinal                                         |
| juures       | рядом               | чуть побольше дистанция. Akna juures on laud                                              |
| **ääres**    | у / вдоль (края)    | **Seina ääres** (У стены). Применяется к объектам где есть край, окно, стена, море, озеро |
| lähedal      | в пешей доступности | Kool on lähedal.                                                                          |
| kaugel       | далеко              | Kino on kaugel                                                                            |
| vasakul      | слева               | Aken on vasakul.                                                                          |
| paremal      | справа              | Paremal on hele tool                                                                      |
| järel        | после / вслед       | **Koer jookseb auto järel.** (Собака бежит за машиной — вслед за ней).                    |
| vastu        | против              | Rohi köha vastu - Лекарство против кашля.                                                 |

### Форма слова
```
põhisõna (omastav) + tagasõna
```


### Построение. Акцент на предмете
Акцент на предмете. **ЧТО** находится?

#### Положительная форма 

**Tagasõna** on V1  
Vasakul on aken. (Слева - окно.)

**V2 tagasõna** **on** V1  
Akna ees on kardinad. (Перед окном - гардина)

**V2-l** on V1 (на поверхности)  
Põranda**l** on hele vaip. (На полу - светлый ковер)  
Laua**l** on tass. (На столе чашка)  

**V2-st слева/справа** on V1  
Laua**st** vasakul on tool. (Слева от стола - стул)

####  Отрицательная форма: V1 => V3

**V2 tagasõna** **ei ole** V3  
Akna ees ei ole kardinat. (Перед окном нет гардины)

**V2-l** ei ole V3 (на поверхности)  
Põrandal ei ole vaipa. (На полу нет ковра)


### Построение. Акцент на месте

Акцент на месте. **ГДЕ** предмет?

V1 on **tagasõna**    
Aken on vasakul. (Окно - слева)  
Kardinad ei ole akna taga   

V1 on **V2 tagasõna**    
Kardinad on akna ees (Гардина - перед окном)  

V1 on **V2-l** (на поверхности)  
Hele vaip on põranda**l**. (Светлый ковер - на полу)  
Tass on laua**l**. (Чашка - на столе)  
Pilt on seina**l**. (Картина на стене)  

V1 on **V2-st слева/справа**  
Tool on laua**st** vasakul (Стул - слева от стола)

## Право / лево

### Статика. Где? (**-l**)

Окончание **-l** говорит о том, что предмет уже там **находится**.  
- **Auto on paremal.** (Машина - справа).      
- **Vasakul on aken.** (Справа — окно).     

Особый случай, когда задействованы два предмета:  
- Laua**st** vasakul on tool. (Слева от стола - стул)  
- Tool on laua**st** paremal - Стул - справа от стола  

### Движение. Куда? (**-le**)

Окончание **-le** говорит о **направлении** движения.  
- **Pööra paremale!** (Поверни направо!).      
- **Vaata paremale!** (Посмотри направо!).      
- Mine vasakule. (Иди налево)  

### Движение. Откуда? (**-lt**)

Окончание **-lt** показывает направление или точку отсчета.   
- **Tema aken on vasakult teine.** — Его окно второе слева (если считать **от** левого края).   
- Paremalt kolmas uks.  (Третья дверь справа / если считать с правой стороны).  

| **Слово**    | **Вопрос**          | **Пример**             | **Перевод**             |
| ------------ | ------------------- | ---------------------- | ----------------------- |
| **Vasakul**  | **Kus?** (Где?)     | _Karp on vasakul._     | Коробка — слева.        |
| **Paremal**  | **Kus?** (Где?)     | _Sahtel on paremal._   | Ящик — справа.          |
| **Vasakule** | **Kuhu?** (Куда?)   | _Mine vasakule._       | Иди налево.             |
| **Paremale** | **Kuhu?** (Куда?)   | _Vaata paremale._      | Посмотри направо.       |
| **Vasakult** | **Kust?** (Откуда?) | _Võta vasakult teine._ | Возьми второй слева.    |
| **Paremalt** | **Kust?** (Откуда?) | _Pööra paremalt ära._  | Сверни (оттуда) справа. |
