# Вежливость (Viisakus)

## Использование глаголов (-ke, -ge)

При обращении к кому-то в вежливой форме к глаголам добавляется суффикс `-ke` или `-ge`

Сложность в понимании куда какой суффикс. Есть несколько способов (правил)

### **Способ "Пение":**

- Если звук можно тянуть/петь (**гласные** или звонкие **L, M, N, R**), то ставь **-ge**.  
- Если звук короткий, взрывной и его не пропеть (**K, P, T, S**), то ставь **-ke**.

| **Если звук перед окончанием...**  | **Ставим** | **Пример**                  |
| ---------------------------------- | ---------- | --------------------------- |
| **Глухой** (k, p, t, s)            | **-ke**    | Võt**ke**, Ost**ke**        |
| **Гласный** (a, e, i, o, u...)     | **-ge**    | Proovi**ge**, Vabanda**ge** |
| **Звонкий согласный** (l, m, n, r) | **-ge**    | Tul**ge**, Laul**ge**       |

Примеры

| **sina vorm** | **teie vorm** |                                                                      |
| ------------- | ------------- | -------------------------------------------------------------------- |
| vabanda       | vabandage     | заканчивается на гласную **a**                                       |
| proovi        | proovige      | заканчивается на гласную **i**                                       |
| laula         | laulge        | заканчивается на звонкую **l**                                       |
| tule          | tulge         | заканчивается на звонкую **l**                                       |
| too           | tooge         | длинная гласная **oo**                                               |
| joo           | jooge         | длинная гласная **oo**                                               |
| sööma         | sööge         | длинная гласная öö                                                   |
| ole           | olge          | (будьте) слово-исключение. всегда **-ge**                            |
| ära           | ärge          | слово-исключение. вспомогательный глагол отрицания, всегда **-ge**   |
| mine          | minge         | слово-исключение. Несмотря на звонкую «n», здесь всегда **-ge**      |
| vaata         | vaadake       | заканчивается на гласную, **но** в слове _vaatama_ сильная буква «t» |
| anna          | andke         | корень _and-_, но при добавлении окончания звук оглушается           |
| hüppa         | hüpake        | (прыгайте) Глухая **p** требует глухого **-ke**.                     |
| võtta         | võtke         | Глухая **t**                                                         |
| osta          | ostke         | Глухая **t**                                                         |


### **Способ "DA-инфинитив"**

Посмотри на слово в *da-форме*:  
- Если там **-DA**, то вежливая форма — **-ge** (*Söö-da* -> *Söö-ge*).  
- Если там **-TA**, то вежливая форма — **-ke** (*Võt-ta* -> *Võt-ke*).  

## Извините

`Vabanda`. При обращении к близким людям, друзьям  
Vabanda, mis kell on?

`Vabandage`. При обращении в уважительной форме или к незнакомым людям в магазине, школе, больнице и т.п.  
Vabandage, mis kell on? 

`vabandust` - нейтральное
