# Повседневные активности

ärkan üles - просыпаюсь  
tõusen voodist - встаю с кровати  
pesen nägu - умываюсь  
pesen hambaid - чищу зубы  
käin duši all - принимаю душ  
kammin juukseid - расчёсываю волосы  
panen riidesse - одеваюсь  
joon kohvi / teed - пью кофе / чай  
söön hommikusööki - завтракаю  
viin lapse lasteaeda - отвожу ребёнка в сад  
koristan tuba - убираю комнату  
pesen nõusid - мою посуду  
teen süüa - готовлю еду  
söön lõunat - обедаю  
pesen pesu - стираю бельё  
triigin riideid - глажу одежду  
pesen põrandat - мою пол  
viin prügi välja - выношу мусор  
lähen magama - иду спать
