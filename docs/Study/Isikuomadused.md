# Isikuomadused - Личные качества

## Список с антонимами (Vastandsõnad)

aktiivne / passiivne - активный / пассивный  
ambitsioonikas / ambitsioonitu - амбициозный / неамбициозный  
aus / ebaaus - честный / нечестный  
sõbralik / ebasõbralik - дружелюбный / недружелюбный  
täpne / ebatäpne - точный (пунктуальный) / неточный  
viisakas / ebaviisakas - вежливый / невежливый  
organiseeritud / organiseerimatu - организованный / неорганизованный  
tark / rumal - умный / глупый  
tähelepanelik  / hajameelne - внимательный / рассеянный  
tugev / nõrk - сильный / слабый  
töökas / laisk - трудолюбивый / ленивый  
rõõmsameelne / kurvameelne - жизнерадостный / грустный (печальный)  
kohusetundlik / vastutustundetu - добросовестный / безответственный  
iseseisev / abitu - самостоятельный / беспомощный  
hoolikas / hooletu - старательный, аккуратный / небрежный  

## Отрицание
В эстонском языке отрицание часто строится по принципу:
1. **Eba-** (как «не-»): _aus_ (честный) → _ebaaus_ (нечестный).    
2. **-tu** (суффикс отсутствия): _vastutus_ (ответственность) → _vastutustundetu_ (безответственный).

## Должен / не должен
`Peab olema` = positiivne (+)
Treener peab olema aktiivne ja tugev.

`Ei tohi olla` = negatiivne (-)
Treener ei tohi olla passiivne ja nõrk.
