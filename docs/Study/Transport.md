# Транспорт

## Основные термины
**reisija** - пассажир  
**reisijad** - пассажиры  
**tuuleklaas** - лобовое стекло  
**pagasiruum** - багажное отделение  
**salong / bussi salong** - салон / салон автобуса  
**vahekäik** - проход между сиденьями  
**iste** - сиденье  
**istmed** - сиденья / кресла  
**seljatugi** - спинка сиденья  
**peatugi** - подголовник  
**käetugi** - подлокотник  
**polster / polsterdus** - обивка  
**käsipuu** - поручень  
**käepide** - ручка / держатель  
**turvavöö** - ремень безопасности  
**uks** - дверь  
**aken** - окно  
**aste** - ступенька  
**põrand** - пол  
**stoppnupp** - кнопка остановки  
**validaator** - валидатор  
**bussijuht** - водитель автобуса  


## Оснащение и навигация
**pagasiriiul** - багажная полка  
**sõidupilet / pilet** - проездной билет / билет  
**ühistransport** - общественный транспорт  
**kohver** - чемодан  
**sõiduplaan** - расписание движения  
**piletiautomaat** - билетный автомат  
**ühiskaart** - транспортная карта  
**kaardilugeja** - считыватель карт  
**rool** - руль  
**küljepeegel** - боковое зеркало  
**turvaväljapääs** - аварийный выход  
**hädaabinupp** - аварийная кнопка  
**kliimaseade** - кондиционер  
**sisenemine** - вход / посадка  
**väljumine** - выход / высадка  
**järgmine peatus** - следующая остановка  
**lõpp-peatus** - конечная остановка  


## Полезные фразы
**Palun kinnitage turvavöö.** - Пожалуйста, пристегните ремень безопасности.  
**Väljumiseks vajutage nuppu.** - Для выхода нажмите кнопку.  
**Järgmine peatus on ...** - Следующая остановка ...
