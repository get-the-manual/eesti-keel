# Местоимение 'see' этот, эта, это

Указательное местоимение

## Основные формы

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **see** | это / этот / эта | **See** on minu ratas. (Это мой велосипед). |
| **selle** | этого / этой | **Selle** kauba hind. (Цена этого товара). |
| **seda** | это / этого | Ma tahan **seda** osta. (Я хочу это купить). |


## Все падежи (от основы *selle-*)

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **sellesse** | в это (куда?) | Ma usun **sellesse**. (Я верю в это). |
| **selles** | в этом (где?) | **Selles** keldris on pime. (В этом подвале темно). |
| **sellest** | из этого / об этом | Ma räägin **sellest**. (Я говорю об этом). |
| **sellele** | этому / на это | Pane raamat **sellele** lauale. (Положи книгу на этот стол). |
| **sellel** | у этого / на этом | **Sellel** rattal on korv. (У этого велика есть корзина). |
| **sellelt** | от этого / с этого | Võta tolm **sellelt** kapilt. (Вытри пыль с этого шкафа). |
| **selleks** | для этого / этим | **Selleks** on vaja aega. (Для этого нужно время). |
| **selleni** | до этого | Ma ei jõudnud **selleni**. (Я до этого не дошел). |
| **sellena** | в качестве этого | **Sellena** see ei toimi. (В качестве этого это не работает). |
| **selleta** | без этого | Ma ei saa **selleta**. (Я не могу без этого). |
| **sellega** | с этим / этим | **Sellega** on kõik korras. (С этим всё в порядке). |
