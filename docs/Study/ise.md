# Местоимение `ise` (сам)

Формы этого местоимения часто используются для того, чтобы подчеркнуть: действие совершено «самим» человеком или направлено на «себя».

## Основные формы
| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **ise** | сам / сама | Tee ise süüa (Готовь сам еду) |
| **enda** | свой | Ma parandan enda ratast. (Я чиню свой велосипед.)	 |
| **ennast** | себя | Mul on vaja ennast ravida (Мне нужно себя лечить) |
| **endale** | себе | Ma ostan endale uue ratta. (Я покупаю себе новый велосипед.)|


## Падежи (от основы *enda-*)

| Вопрос | Эстонская форма | Перевод | Пример |
| :--- | :--- | :--- | :--- |
| куда? | **endasse** | в себя | *Uskuma endasse* (верить в себя). |
| где? | **endas** | в себе | *Ta on endas kindel* (он уверен в себе). |
| откуда? | **endast** | о себе / из себя | *Räägi endast* (расскажи о себе). |
| кому? | **endale** | себе | *Ostan endale* (куплю себе). |
| у кого? | **endal** | у себя / самого | *Mul endal on...* (у меня у самого есть...). |
| от кого? | **endalt** | от себя / у себя | *Küsi endalt* (спроси у самого себя). |
| кем? | **endaks** | собой | *Jää iseendaks* (оставайся собой). |
| без кого? | **endata** | без себя | *Endata on raske* (без самого себя трудно). |
| с кем? | **endaga** | с собой | *Võta endaga kaasa* (возьми с собой). |
