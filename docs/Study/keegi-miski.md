# Местоимения `keegi` (кто-то), `miski` (что-то)

## `keegi` (кто-то)

**Keegi** — это неопределенное местоимение, которое переводится как **«кто-то»**, **«кто-нибудь»** или **«кто-то один»**.

Оно используется, когда мы говорим о человеке, но не знаем точно, о ком именно, или это не имеет значения.

### Основные формы

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **keegi** | кто-то (кто?) | **Keegi** koputab uksele. (Кто-то стучит в дверь). |
| **kellegi** | чье-то / кого-то | See on **kellegi** ratas. (Это чей-то велосипед). |
| **kedagi** | кого-то | Ma ei näe **kedagi**. (Я никого не вижу). |

---

### Все формы (от основы *kellegi-*)

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **kellessegi** / **kellegisse** | в кого-то | Ta usub **kellegisse**. (Он верит в кого-то). |
| **kelleski** / **kellegis** | в ком-то | **Kellegis** on peidus saladus. (В ком-то скрыта тайна). |
| **kellestki** / **kellegist** | о ком-то / от кого-то | Ma kuulsin seda **kellegist**. (Я слышал это от кого-то). |
| **kellelegi** / **kellegile** | кому-то | Anna see **kellelegi** teisele. (Отдай это кому-то другому). |
| **kellelgi** / **kellegil** | у кого-то | **Kellegil** on kindlasti abi vaja. (Кому-то точно нужна помощь). |
| **kelleltki** / **kellegilt** | от кого-то | Sain **kellegilt** kirja. (Получил от кого-то письмо). |
| **kellegagi** / **kellegiga** | с кем-то | Ta räägib **kellegiga**. (Он разговаривает с кем-то). |
| **kelletagi** / **kellegita** | без кого-то | Ma ei taha **kellegita** minna. (Я не хочу идти без кого-то). |

## `miski` (что-то)

Местоимение **miski** относится к неодушевленным предметам или абстрактным понятиям.

### Основные формы

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **miski** | что-то / что-нибудь | **Miski** kukkus maha. (Что-то упало). |
| **millegi** | чего-то / чье-то | **Millegi** hind on tõusnud. (Цена чего-то выросла). |
| **midagi** | что-то / что-нибудь | Ma tahan **midagi** süüa. (Я хочу что-то поесть). |

---

### Все формы (от основы *millegi-*)

| Форма | Перевод | Пример |
| :--- | :--- | :--- |
| **millegisse** | во что-то | Ma usun **millegisse**. (Я верю во что-то). |
| **millegis** | в чём-то | **Millegis** on viga. (В чём-то есть ошибка). |
| **millestki** | о чём-то / из чего-то | Räägi **millestki** huvitavast. (Расскажи о чём-то интересном). |
| **millelegi** | чему-то / на что-то | Ta loodab **millelegi**. (Он надеется на что-то). |
| **millegil** | на чём-то / у чего-то | **Millegil** on plekk peal. (На чём-то есть пятно). |
| **millelgilt** | с чего-то | Võta see **millegilt**. (Возьми это с чего-то). |
| **millegagi** | чем-то / с чем-то | Ma kirjutan **millegagi**. (Я чем-то пишу). |
| **millegita** | без чего-то | See on **millegita** tühi. (Это без чего-то пустое). |
