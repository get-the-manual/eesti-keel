# Описание экзамена уровня A2

В документе находится перевод с эстонского языка описания экзамена. Приводятся ссылки на примеры заданий.

## Структура экзамена по эстонскому языку уровня А2 (перевод)

| Часть (Osa) | Длительность (Kestus) | Задания (Ülesanded) | Проверяемые навыки (Testitavad oskused) |
| :--- | :--- | :--- | :--- |
| **Письмо (Kirjutamine)** | 30 мин | - Перенос информации (заполнение данных с визитки)<br>- Сообщение (записка, приглашение) или описание на заданную тему | Общее умение писать, общее письменное общение (извещения, сообщения, приглашения; творческое письмо) |
| **Аудирование (Kuulamine)** | 30 мин | - Задание с выбором ответа (из 3-х картинок, чисел или слов)<br>- Задание с выбором ответа (из 2-х вариантов, «правильно/неправильно»)<br>- Задание с кратким ответом (написать краткий ответ)<br>- Задание на соответствие (сопоставление информации) | Общее умение слушать (понимание разговора носителей языка; прослушивание объявлений и инструкций) |
| **Чтение (Lugemine)** | 50 мин | - Задание с выбором ответа (из 3-х вариантов)<br>- Задание с выбором ответа (поиск текста, подходящего к предложению)<br>- Задание на соответствие (поиск пояснения к сообщению или описание, подходящее к месту)<br>- Задание с выбором ответа (из 3-х вариантов)<br>- Задание на заполнение пропусков (выбор из 3-х вариантов) | Общее умение читать (чтение переписки; целенаправленное чтение; чтение для получения информации; чтение инструкций) |
| **Говорение (Rääkimine)** | 15 мин | - Описание картинки и ответы на вопросы по теме картинки<br>- Постановка вопросов и ответы на вопросы при помощи карточки с идеями (ideekaart) | Общее умение говорить, общее устное общение, короткий монолог (обмен информацией, роль спрашивающего и отвечающего, описание) |

## Письмо
Тест по письму состоит из двух заданий, на выполнение каждого из которых отводится 30 минут.

**Первое задание** [`A2_Kirjutamine_Esimene_ülesanne2.pdf`](https://harno.ee/sites/default/files/documents/2021-06/A2_Kirjutamine_Esimene_%C3%BClesanne2.pdf):  
Задача состоит в том, чтобы записать данные с визитной карточки (участник теста должен использовать предоставленные данные). Текст должен содержать не менее 25 слов.

**Во втором задании** испытуемый должен написать либо  
а) объявление, короткое сообщение или приглашение [`A2_Kirjutamine_Teine_ülesanne_A_teade. lühisõnum.pdf`](https://harno.ee/sites/default/files/documents/2021-06/A2_Kirjutamine_Teine_%C3%BClesanne_A_teade.%20l%C3%BChis%C3%B5num.pdf). Текст должен содержать не менее 30 слов, или  
б) творческий текст [`A2_Kirjutamine_Teine_ülesanne_B_loovtekst.pdf`](https://harno.ee/sites/default/files/documents/2021-06/A2_Kirjutamine_Teine_%C3%BClesanne_B_loovtekst.pdf). (Описание на заданную тему). Текст должен содержать не менее 30 слов.


## Слушание
Тест на аудирование состоит из четырех заданий и длится 30 минут. В тесте используются сокращенные, адаптированные, упрощенные и/или специально написанные тексты. Текст для каждого задания прослушивается дважды. Перед каждым заданием предусмотрена пауза, чтобы испытуемый мог ознакомиться с вопросами, прежде чем приступить к выполнению задания.

### **Первое задание**
[`A2_Kuulamine_esimene_ülesanne.pdf`](https://harno.ee/sites/default/files/documents/2024-03/A2_Kuulamine_esimene_%C3%BClesanne.pdf)  представляет собой тест с несколькими вариантами ответа, состоящий из семи вопросов. Испытуемый прослушивает короткие речевые фрагменты (монологи) продолжительностью примерно 20-30 секунд. Ответы необходимо выбрать из трех вариантов: изобразительный, словесный или числовой.

Источником текста может быть:
- Личные и публичные объявления и инструкции;  
- краткие описания людей, вещей, мест, видов деятельности;  
- развлечение;  
- спортивные комментарии.

Прослушайте аудиофайл, являющийся примером выполнения первого задания [`Kuulamisülesanne nr 1 A2.mp3`](https://projektid.edu.ee/spaces/THO/pages/343705183/Konsultatsioonide+materjalid?preview=/343705183/343705193/Kuulamis%C3%BClesanne%20nr%201%20A2.mp3).
