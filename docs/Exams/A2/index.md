# Экзамен на уровень А2

Информация по уровневым экзаменам по эстонскому языку находится на сайт [harno.ee](https://harno.ee).  
Сайт на трех языках: эстонском, английском и русском. Причем самая полная информация находится в эстоноязычной версии.

## Страница на эстонском языке

Открываем страницу 
[Eesti keele tasemeeksamid | Haridus- ja Noorteamet](https://harno.ee/eesti-keele-tasemeeksamid)

Для изучения информации по подготовке к экзамену идем по линке или листаем до `Eksamiks valmistumine` (Подготовка к экзамену).

Смотрим в раздел `Eesti keele A2-taseme eksam` (Экзамен по эстонскому языку уровня А2)

## EKSAMIKS VALMISTUMINE

### Полезные ресурсы:

[Keeletee](https://www.keeletee.ee/) – онлайн-курсы для изучения эстонского языка

[Keeleklikk](https://www.keeleklikk.ee/et/welcome) – Бесплатный виртуальный курс эстонского языка для начинающих, проводимый на русском или английском языке.

[Veebipõhised eesti keele tasemetestid](http://web.meis.ee/testest/) – Онлайн-тесты на знание эстонского языка. Проверьте свои языковые навыки перед сдачей экзамена.
