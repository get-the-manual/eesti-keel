# Kuulamine. Esimene Ülesanne

Испытуемый прослушивает короткие речевые фрагменты (монологи) продолжительностью примерно 20-30 секунд. Ответы необходимо выбрать из трех вариантов: изобразительный, словесный или числовой.

## Пример 1 (harno.ee)  

[Аудио `Kuulamisülesanne nr 1 A2.mp3`](https://projektid.edu.ee/spaces/THO/pages/343705183/Konsultatsioonide+materjalid?preview=/343705183/343705193/Kuulamis%C3%BClesanne%20nr%201%20A2.mp3)

[Бланк с вопросами: `A2_Kuulamine_esimene_ülesanne.pdf`](https://harno.ee/sites/default/files/documents/2024-03/A2_Kuulamine_esimene_%C3%BClesanne.pdf)
